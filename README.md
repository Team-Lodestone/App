# Lodestone.App
The frontend for Project Lodestone.
Uses [libLodestone](https://github.com/Team-Lodestone/libLodestone) internally.

## Roadmap
- [ ] Allow for loading a World, Level, maybe even individual regions and chunks.
- [ ] Allow for exporting worlds, levels, chunks, etc
  - [ ] Allow for exporting e.g. a Level as a World. 
  - This is how we will achieve user facing world conversion.
- [ ] Translations
